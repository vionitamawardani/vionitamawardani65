Pengalaman Selama Pembelajaran Jarak Jauh

Pada pembelajaran jarak jauh awalnya saya merasa sedikit senang karena saya tidak harus pulang pergi kesekolah, berangkat di pagi hari buta dan pulang menjelang petang. Tetapi lama kelamaan saya mulai merasa bosan dan saya mulai banyak mengalami kesulitan saat memahami materi, karena saat pembelajaran secara online saya sulit saat bertanya jika tidak mengerti sesuatu bahkan terkadang sudah dijelaskan kembai tapi saya merasa masih belum paham apabila tidak dijelaskan secara langsung.

Saat pembelajaran secara online saya memiliki bnayak sekali pengaaman yang tidak mengenakan, salah satunya adalah saya pernah mengirim satu file tugas yang salah ke google classroom dan ahasil saya mendapatkan nilai 60 padahal sebenarnya saya sudah mengerjakan tugas tersebut dengan benar dan lengkap, akhirnya saya menghubungi guru yang bersangkutan untuk menjelaskan apa yang sebenarnya terjadi dan meminta maaf atas keteledoran saya, setelah saya mengirim ulang tugasnya guru tersebut kemudian memaafkan saya dan memberi saya niai 95.
